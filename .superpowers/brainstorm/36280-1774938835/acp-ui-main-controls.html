ACP（方案 B2）：主区控制条（流式输出上的操作）

选一个“控制条组织方式”。目标：不打断阅读、但关键操作随手可达。

C1

单行工具条（紧凑）

一行放齐：滚动、尾部、搜索、复制。适合熟练用户。

预览：工具条
输出区（滚动）

优点

  • 最快
  • 最省空间

缺点

  • 按钮密集
  • 新手不易理解“只看尾部”
C2

分组工具条（推荐）

按心智模型分组：阅读（滚动/尾部）→ 查找（搜索）→ 导出（复制）。

预览：分组
阅读
查找
导出
输出区（滚动）

优点

  • 可发现性好
  • 不容易误触

缺点

  • 占用更宽
  • 移动端需要折叠
C3

极简 + 溢出菜单

只保留 2 个高频按钮（暂停/尾部），其余进 “更多”。

预览：极简
（更多里：搜索/复制/导出）
输出区（滚动）

优点

  • 最干净
  • 适合窄屏

缺点

  • 二次点击成本
  • 搜索不够显眼
