# AI Agent 对话系统 - 快速使用指南

## 🚀 快速开始

### 方式一:一键快速开始(推荐)

1. 启动应用:`make dev`
2. 打开应用,点击左侧导航 "🤖 AI Agent 对话"
3. 点击页面中央的 **"⚡ 快速开始"** 按钮
4. 系统会自动创建一个默认 Agent,但需要你配置 API Key
5. 开始聊天!

### 方式二:手动创建 Agent

1. 点击右上角 **"➕ 创建 Agent"** 按钮
2. 填写表单:
   - **名称**: 给你的 Agent 起个名字
   - **类型**: 选择 main/sub/worker
   - **Base URL**: 默认 OpenAI API,也可以用其他兼容服务
   - **API Key**: 必填! 你的 OpenAI API Key
   - **模型**: 如 gpt-4o、gpt-3.5-turbo
   - **系统提示**: 自定义 Agent 行为
3. 点击创建
4. 从左侧列表选中刚创建的 Agent
5. 开始对话!

## 🎯 核心功能

### 1. 项目理解

Agent 可以:
- ✅ 自动识别项目语言和结构
- ✅ 搜索代码文件
- ✅ 读取文件内容
- ✅ 统计代码行数

示例对话:
```
你: 这个项目是做什么的?
AI: 这是一个基于 Wails + Go + React 的桌面应用...

你: 搜索包含 "Agent" 的代码
AI: 找到以下文件包含 "Agent":
    - backend/internal/agent/agent.go
    - frontend/src/modules/agent/...
```

### 2. 多 Agent 协作

- **主代理**(main): 顶层对话,可以创建子代理
- **子代理**(sub): 专门处理特定任务
- **工作代理**(worker): 执行具体工作

### 3. 技能系统

Agent 可以启用技能(需要在创建时配置):
- `create-rule`: 创建 Cursor 规则
- `feishu-doc`: 飞书文档操作
- 其他自定义技能...

### 4. Agent 状态

- 🟢 **空闲**(idle): 可以接收新消息
- 🟡 **繁忙**(busy): 正在处理请求
- ⚫ **已停止**(stopped): Agent 已销毁

## ⚙️ 高级配置

### 模型参数

- **Max Tokens**: 单次回复最大长度(建议 2048-4096)
- **Temperature**: 创造性(0-2,推荐 0.7)
  - 0 = 最确定性
  - 1 = 平衡
  - 2 = 最随机

### API Key 配置

**方式一**: 在创建 Agent 时直接填写

**方式二**: 统一管理(推荐)
1. 点击左侧导航 "模型管理"
2. 添加你的 API Key
3. 所有 Agent 都可以使用

### 使用其他 API 服务

本系统支持任何 OpenAI 兼容的 API:

- **OpenAI**: `https://api.openai.com/v1`
- **DeepSeek**: `https://api.deepseek.com/v1`
- **本地 Ollama**: `http://localhost:11434/v1`
- **其他代理服务**: 填写对应 Base URL

## ❓ 常见问题

### Q: 为什么看不到聊天输入框?

**A**: 需要先创建并选中一个 Agent。点击"快速开始"或"创建 Agent"。

### Q: 发送消息没反应?

**A**: 检查:
1. 是否填写了 API Key
2. API Key 是否正确
3. 网络连接是否正常
4. 查看右下角是否有错误提示

### Q: 如何删除 Agent?

**A**: 在左侧 Agent 列表中,鼠标悬停到 Agent 上,点击"删除"按钮。

### Q: 可以同时和多个 Agent 对话吗?

**A**: 可以! 创建多个 Agent,在左侧列表切换即可。每个 Agent 有独立的对话历史。

### Q: 对话历史会保存吗?

**A**: 当前版本对话历史保存在内存中,关闭应用后会清空。后续版本会增加持久化。

## 🔧 故障排除

### 编译错误

如果启动时遇到编译错误:

```bash
# 清理重新编译
make clean
make dev
```

### API 连接超时

如果 API 请求超时:
1. 检查网络连接
2. 尝试使用代理
3. 增加 Max Tokens 设置中的超时时间

### 界面显示异常

刷新页面或重启应用。
