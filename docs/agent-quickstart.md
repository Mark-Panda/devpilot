# AI Agent 对话 - 快速开始指南

## 🎯 第一次使用

### 步骤 1: 配置模型

1. 点击侧边栏的 **"模型管理"**
2. 点击 **"添加模型配置"**
3. 填写信息:
   - **Base URL**: 例如 `https://api.openai.com/v1`
   - **API Key**: 你的 API 密钥
   - **站点描述**: 例如 "OpenAI"
   - **模型列表**: 例如 `gpt-4o`, `gpt-3.5-turbo` (每行一个)
4. 保存配置

### 步骤 2: 开始对话

1. 点击侧边栏的 **"AI Agent 对话"**
2. 在欢迎界面选择一个模型
3. 点击 **"开始对话 →"** 按钮
4. 在输入框输入消息,按 Enter 发送!

- **Shift + Enter**: 换行

## 🔄 切换模型

如果想换个模型:
1. 点击右上角的 **"切换模型"** 按钮
2. 选择其他模型
3. 点击"开始对话"重新开始

## ⚠️ 常见问题

### Q: 点击"开始对话"后显示错误?
**A**: 检查你的 API Key 是否正确,Base URL 是否可访问

### Q: 没有可选的模型?
**A**: 需要先在"模型管理"中添加至少一个模型配置

### Q: AI 回复很慢?
**A**: 可能是网络问题或 API 服务器响应慢,请耐心等待

### Q: 想要创建多个 Agent?
**A**: 目前自动创建一个主 Agent,未来版本会支持多 Agent 管理

## 🎨 界面说明

### 欢迎界面
- 🤖 **大图标**: AI Agent 标识
- 🎯 **模型选择器**: 选择要使用的 AI 模型
- ✅ **勾选标记**: 显示当前选中的模型
- 🚀 **开始按钮**: 创建 Agent 并开始对话

### 对话界面
- **顶部栏**: 显示 Agent 名称和模型
- **状态徽章**: 绿色表示在线
- **消息区**: 
  - 👤 蓝色气泡 = 你的消息
  - 🤖 白色气泡 = AI 回复
- **输入框**: 底部输入区域
- **字符计数**: 右下角显示字符数
