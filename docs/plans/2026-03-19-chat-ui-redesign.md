# Chat UI Redesign — 对齐 OpenClaw 参考界面

**日期**：2026-03-19  
**范围**：`Layout.tsx`、`AgentChatPage.tsx`、`ChatMessages.tsx`、`globals.css`

---

## 目标

将 DevPilot 聊天界面的视觉结构对齐参考截图（OpenClaw 风格），保留现有菜单项与功能逻辑，不引入新依赖。

---

## 改动范围

### 1. Layout.tsx — 左侧导航分组 + 版本号

**现状**：菜单项平铺，无分组，无版本号。

**目标**：
- 菜单项按功能分组，加分组标题（小字灰色大写）：
  - *(无标题)* → 聊天
  - **控制** → 重构路由管理、接口对比、终端
  - **规则引擎** → RuleGo 规则管理、RuleGo 执行日志
  - **设置** → 技能仓库、模型管理
- 左侧底部加版本号（`版本 vX.X.X`，灰色小字 + 绿色状态点）

**数据结构变更**：`menuItems` 改为带 `group` 字段的分组结构。

---

### 2. AgentChatPage.tsx — 顶部工具栏重构

**现状**：面包屑 + 两个 `<select>` 下拉框 + 4 个图标按钮，两行布局。

**目标**：单行工具栏，从左到右：
1. **汉堡菜单图标**（预留，暂不实现折叠功能）
2. **面包屑**：`DevPilot > 聊天`
3. **Agent pill**：圆角胶囊，显示当前 agent 名，点击可切换（下拉）
4. **Model pill**：圆角胶囊，显示当前模型名
5. **右侧图标组**：搜索（⌘K）、刷新、停止、历史、主题、设置

移除原有的第二行配置栏（`flex-wrap items-center gap-3 py-2 mb-2 border-b`）。

---

### 3. ChatMessages.tsx — Tool output 卡片样式

**现状**：工具调用以内联小按钮形式展示（`text-xs px-2.5 py-1.5 rounded-lg border`）。

**目标**：每个工具调用渲染为独立可折叠卡片：
```
┌─────────────────────────────────┐
│  • ✦ {tool.name}  read          │  ← 点击展开/收起
│  [展开后显示 summary]            │
└─────────────────────────────────┘
```
- 卡片样式：白色背景、细边框、圆角、左侧有小 `✦` 图标
- 默认收起，点击展开 summary

---

### 4. globals.css — 导航分组标题样式

新增：
- `.app-nav-group-label`：分组标题，小字、灰色、大写、字间距
- `.app-version`：底部版本号区域样式

---

## 不改动

- `ChatInput.tsx`：功能与样式基本符合参考图，保持不变
- 其他页面（路由管理、接口对比等）：不受影响
- 后端逻辑：零改动

---

## 成功标准

1. 左侧导航有分组标题，底部有版本号
2. 聊天页顶部为单行工具栏，agent/model 为 pill 样式
3. Tool output 为独立可折叠卡片
4. 视觉风格与参考截图一致
